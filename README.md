# Movie Watchlist

A beautiful, responsive web application that fetches and displays movies from the Studio Ghibli API. 

## Features
- **Dynamic Data**: Automatically fetches films via the Studio Ghibli API.
- **Smart Filtering**: Instantly sort and filter movies by Director or Release Year.
- **Responsive Layout**: Fluid CSS Grid that adapts perfectly to desktop, tablet, and mobile screens.
- **Modern UI**: A cinematic dark theme featuring glass-like aesthetics and smooth hover animations.

## Technologies Used
- HTML5
- CSS3 (CSS Grid, Flexbox, Media Queries)
- JavaScript
